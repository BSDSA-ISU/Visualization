# 📊 Types of Data & Levels of Measurement — Reviewer

---

## 🧠 I. Types of Data

### 1. **Qualitative (Categorical)**

Describes qualities or characteristics. Non-numeric.

| Type | Description | Examples |
| ----- | ------------- | ---------- |
| **Nominal** | Labels with no order | Gender, Blood type, Nationality |
| **Ordinal** | Ordered categories | Rankings, Satisfaction level, Class standing |

---

### 2. **Quantitative (Numerical)**

Represents quantities. Numeric values.

| Type | Description | Examples |
|-----|-------------|----------|
| **Discrete** | Countable integers | Number of students, Cars, Pets |
| **Continuous** | Measurable values | Height, Weight, Time |

---

## 📏 II. Levels of Measurement

| Level | Has Order? | Equal Intervals? | True Zero? | Examples |
|------|-------------|------------------|------------|----------|
| **Nominal** | ❌ | ❌ | ❌ | Gender, Color |
| **Ordinal** | ✔ | ❌ | ❌ | Rankings, Grades (A–F) |
| **Interval** | ✔ | ✔ | ❌ | Temperature (°C/°F), IQ |
| **Ratio** | ✔ | ✔ | ✔ | Height, Age, Income |

---

## 🔑 Key Differences Shortcut

**NOIR Rule (Ascending strength):**

> **N**ominal → **O**rdinal → **I**nterval → **R**atio

Each level adds one property:
- Nominal → categories
- Ordinal → + order
- Interval → + equal spacing
- Ratio → + true zero

---

## 🎯 Quick Identification Guide

Ask these questions in order:

1. Is it just a label? → **Nominal**
2. Does order matter? → **Ordinal**
3. Are differences meaningful? → **Interval**
4. Can zero mean “none”? → **Ratio**

---

## 🧪 Practice Examples

| Variable | Type | Level |
|--------|------|------|
| Zip Code | Qualitative | Nominal |
| Race Position | Qualitative | Ordinal |
| Temperature (°C) | Quantitative | Interval |
| Number of Books | Quantitative | Ratio |

---

## ⚡ Memory Tricks

- **Nominal = Name**
- **Ordinal = Order**
- **Interval = Intervals matter**
- **Ratio = Real zero**
